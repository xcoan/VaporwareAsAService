Vaporware As A Service

Welcome to Vaporware As a Service... as a service.

Options:
'/app' will generate an idea for a groundbreaking new application.
'/app/json' will return a JSON response.
The response is a JSON object with the following keys: 'appname' and 'description'

'/update' will generate an update for you to share with friends, backers, stakeholders, etc.
'/update/json' will return a JSON response.
The response is a JSON object with the following keys: 'title' and 'body'.
